# AB Switch Widget

Android widget za prikaz izmen A/B.

## Funkcije
- prikaz trenutne izmene
- prilagodljiv prefix
- prilagodljive oznake A/B/X
- samodejna posodobitev ob polnoči

## Namestitev
Aplikacija je na Google Play:
https://play.google.com/store/apps/details?id=com.dante.abworkdaywidget

## Razvoj
Projekt je narejen v:
- Android Studio
- Kotlin
- minSdk 26
